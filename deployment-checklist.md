# Deployment Checklist

## Before launch

- [ ] Replace email address
- [ ] Replace LinkedIn profile URL
- [ ] Add a real professional photo
- [ ] Replace placeholder video URL
- [ ] Decide whether the contact form should use mailto, Netlify Forms or Formspree
- [ ] Check employer policy on public content
- [ ] Remove or anonymise any customer-sensitive details
- [ ] Test on mobile, tablet and desktop
- [ ] Run through all navigation links
- [ ] Add Google Analytics, Plausible or Fathom if you want visitor analytics

## SEO basics

- [ ] Buy a simple domain, e.g. `thomasjacques.co.uk`, `tomjacques.com`, or similar
- [ ] Set the page title and meta description
- [ ] Add Open Graph image
- [ ] Submit sitemap later if the site grows
- [ ] Add schema markup later if you add speaking, consulting or article content

## Suggested hosting route

Fastest professional route:

1. Create a GitHub repository
2. Upload all files
3. Connect repository to Netlify or Cloudflare Pages
4. Point your domain at the hosting provider
5. Enable HTTPS

## Content rhythm

For the first 8 weeks:

- 1 short video per week
- 1 article every 2 weeks
- 2 short LinkedIn posts per week pointing back to the site
- 1 technical diagram or field note per week
